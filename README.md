# vigilant-chip8
Simple chip8 emulator in c++

## Run with
'''
./Vigilant-chip8 [path to chip8 program]
'''
## Debug mode allows you to step through the emulation.
'''
./Vigilant-chip8 --debugger [path to chip8 program]
